# Student Course Management System

A simple full-stack project that allows admins to manage courses and students to enroll.

## Features
- Add and view courses
- Student enrollment
- REST API design
- Frontend demo using GitHub Pages

## Tech Stack
- Java, Spring Boot
- MySQL
- HTML, JavaScript
